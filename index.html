一、默认初始化：

$('#demo1').citySelect();

二、按行政区划代码初始化：

$('#demo2').citySelect({search: 441900401});  // 广东省东莞市松山湖管委会

三、按地址名称初始化：

$('#demo3').citySelect({search: '香港特别行政区,香港岛,湾仔区'});

四、下拉框 change 回调函数：当下拉框的值改变时触发，返回当前下拉框的值和对应文字

$('#demo4').citySelect({
    onchange: function(code, name) {
        $('#result1').html('当前选择：' + name + '(' + code + ')');
    }
});

五、设置缺省提示语：

$('#demo5').citySelect({prompt: ['- 省/直辖市 -', '- 市 -', '- 区/县 -', '- 乡/镇/街道办 -']});

六、取值：

$('#fetch').click(function() {
    $('#demo5').citySelect(function(code, nameArr) {
        $('#result2').html(
            '行政代码：' + code + '<br>' +
            '地址：' + nameArr.join('，')
        );
    });
});
