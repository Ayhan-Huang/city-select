一、默认初始化：

$('#demo1').citySelect();

二、按行政区划 code 初始化：

$('#demo2').citySelect({search: 441900401});  // 广东省东莞市东莞市松山湖管委会

三、按地址名称初始化：

$('#demo3').citySelect({search: '广东省,深圳市,福田区'});

四、回调函数：

$('#demo4').citySelect({
    onchange: function(code, name) {
        $('#result').html('当前选择：' + name + '(' + code + ')');
    }
});

五、设置缺省提示语：

$('#demo5').citySelect({prompt: ['- 省/直辖市 -','- 市 -', '- 区/县 -', '- 乡镇/街道办 -']});
